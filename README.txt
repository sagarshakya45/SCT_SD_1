Temperature Converter

This is a simple C++ program that changes temperature from one unit to another.

What it can do:
- Change Celsius to Fahrenheit and Kelvin
- Change Fahrenheit to Celsius and Kelvin
- Change Kelvin to Celsius and Fahrenheit
- Shows numbers with 2 decimal places

How to use:
1. Run the program.
2. Choose the unit of your temperature:
   - Enter 1 for Celsius
   - Enter 2 for Fahrenheit
   - Enter 3 for Kelvin
3. Enter the temperature number.
4. See the converted temperatures.

Example:
Welcome to the Temperature Converter!
What unit is your temperature in?
1. Celsius
2. Fahrenheit
3. Kelvin
Please enter 1, 2, or 3:
1
Enter the temperature value: 25
You entered: 25.00 Celsius
In Fahrenheit: 77.00 Fahrenheit
In Kelvin: 298.15 Kelvin

Thank you for using the converter!

What you need:
- A C++ compiler (like g++)
- The program uses standard C++ libraries

How to run:
If you use g++, type these commands:

g++ -o TemperatureConvertor TemperatureConvertor.cpp
./TemperatureConvertor
